# Getting Started

## Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

## Setup & Running

1. **Configure environment** — Copy or edit the `.env` file and fill in the required values.
2. **Extract build artifacts** — Unzip the `.next.zip` file into the project root.
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Start the server**
   ```bash
   npx next start
   ```
